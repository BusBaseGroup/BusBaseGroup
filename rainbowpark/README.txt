GITHUB-ONLY MONTHLY STAFF ROTA — REDESIGNED

FILES
- maker.html
- viewer.html
- config.js
- rota-data.json

HOW TO USE
1. Upload all files to the same GitHub Pages folder.
2. Open maker.html.
3. Use the Staff tab to add staff.
4. Use the Rides tab to add rides and choose how many staff each ride needs.
5. Open the Monthly rota tab.
6. Press Manage days off beside any date.
7. Tick staff who need that day off.
8. Replacement staff are assigned automatically.
9. Press Download rota-data.json at the bottom.
10. Replace the existing rota-data.json in GitHub with the downloaded file.

The public viewer page reads rota-data.json automatically.
